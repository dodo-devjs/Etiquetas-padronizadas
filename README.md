# 🏷️ Viva Art – Sistema de Impressão de Etiquetas

Sistema web simples para geração e impressão de etiquetas personalizadas, desenvolvido para uso interno da Viva Art Armarinhos.

O sistema permite cadastrar produtos, organizar etiquetas em uma folha A4 e imprimir apenas a área necessária, reduzindo desperdício de papel.

---

## 🚀 Funcionalidades

- Cadastro de:
  - Nome do Produto
  - Valor
  - Quantidade

- Modelos de etiqueta:
  - **Tamanho 1:** 10,5cm x 1,46cm (Etiqueta padrão)
  - **Tamanho 2:** 20cm x 9,15cm (Etiqueta promocional)

- Acúmulo de etiquetas na folha antes da impressão
- Organização automática em formato A4
- Impressão apenas da área das etiquetas
- Interface personalizada
- Etiquetas em preto para melhor qualidade de impressão

---

## 🖥️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Layout com medidas reais em centímetros (cm)

---

## 📦 Como Utilizar

1. Baixe o arquivo `etiquetas.html`
2. Abra no navegador (recomendado: Google Chrome)
3. Preencha os campos:
   - Nome do Produto
   - Valor
   - Quantidade
   - Selecione o tamanho
4. Clique em **Adicionar à Folha**
5. Após organizar as etiquetas, clique em **Imprimir Folha**

---

## 🖨️ Configuração Recomendada de Impressão

- Papel: A4
- Escala: 100%
- Margens: Nenhuma ou mínima
- Desativar cabeçalhos e rodapés do navegador
